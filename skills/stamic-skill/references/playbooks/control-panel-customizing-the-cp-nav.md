# Control Panel: Customizing the CP Navigation

**Summary:** Users can customize the CP sidebar navigation in Preferences. With Statamic Pro and appropriate permission, you can configure defaults for other users/roles. Addons can also extend navigation.

**When to use:**
- Helping editors focus (hide items, reorder).
- Setting default CP nav for roles.

## Steps
1. Users manage their own nav via CP: Settings â†’ Preferences.
2. To edit defaults for others:
   - enable Statamic Pro
   - be super user or have permission to manage preferences
3. For addons, use the CP navigation extension helpers (see CP nav extending docs).
