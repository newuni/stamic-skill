# Control Panel: Multi-User Collaboration

**Summary:** Multi-user collaboration adds real-time presence + field locking while multiple users edit the same entry.

**When to use:**
- Editorial teams editing concurrently.

## Steps
1. Install the Collaboration addon:
   - https://statamic.com/addons/statamic/collaboration
2. Confirm features:
   - presence indicators
   - per-field locking
   - real-time updates
