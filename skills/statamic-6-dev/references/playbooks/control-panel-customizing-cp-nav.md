# Control Panel: Customizing the CP Navigation

**Summary:** Users can customize the CP sidebar navigation in Preferences. With Pro + permissions, admins can set defaults per role or globally. Addons can also extend CP nav via PHP helpers.

**When to use:**
- Simplifying the CP for editors/clients.
- Setting consistent defaults across roles.

## Steps
1. Users: adjust CP nav under Settings → Preferences.
2. Admins: enable Pro + ensure permissions to manage preferences, then set defaults globally or per role.
3. For addon-driven nav items, use Statamic’s CP nav extension helpers (addon layer).

## Pitfalls / gotchas
- CP navigation customization is different from frontend “Navs” content structures.
